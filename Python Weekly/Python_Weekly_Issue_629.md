# 最新发布  
  
[Django 5.0 已发布](https://www.djangoproject.com/weblog/2023/dec/04/django-50-released/)  
发行说明详细介绍了大量令人兴奋的新功能，其中一些亮点是：
  * 数据库计算的默认值：允许将数据库计算的默认值定义为模型字段。
  * 延续 Django ORM 扩展的趋势，生成的模型字段允许创建数据库生成的列。
  * 字段组的概念被添加到模板系统中，以简化表单字段呈现。
